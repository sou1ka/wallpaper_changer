# Wallpaper Changer

起動中に壁紙を自動切換えするアプリです。  
アプリを終了、または設定範囲から外れるとアプリ起動時の壁紙を復元します。

## 機能

**画像設定**

「画像を参照」するか、「一覧」に画像をドラッグアンドドロップすると切り替え用の画像が追加されます。  
追加した画像をダブルクリックすると削除されます。

**切替時間**
壁紙を切り替える時間を秒数で指定します。

**ラムダム表示する**

追加した画像をランダム壁紙にする場合はチェックしてください。

**開始時刻**

壁紙切り替えを開始する時刻を設定します。  
未設定の場合はアプリ起動中に壁紙が切り替わります。

**終了時刻**

壁紙切り替えを終了する時刻を設定します。  
終了時刻を超えるとアプリ起動時の壁紙を復元します。  
未設定の場合はアプリ起動中は終了しません。

**曜日**

壁紙切り替えを実行する曜日を設定します。  
未設定の場合はアプリ起動中に壁紙が切り替わります。

**日付**

壁紙切り替えを実行する日を設定します。  
未設定の場合はアプリ起動中に壁紙が切り替わります。

---

UIは Windows UI を利用しています。  
https://windows-ui.github.io/

# Wallpaper Changer

An app that automatically switches wallpapers while running.
When the app exits, or when the current time falls outside the configured range, the wallpaper at the time the app was launched is restored.

## Features

**Image Settings**

Click "Browse Images" or drag and drop images into the list to add them as wallpaper candidates.  
Double-clicking an added image removes it.

**Switch Interval**

Specifies how often (in seconds) the wallpaper is changed.

**Random Display**

Check this option to display the added images in random order.

**Start Time**

Sets the time at which wallpaper switching begins.  
If not set, wallpapers will switch whenever the app is running.

**End Time**

Sets the time at which wallpaper switching ends.  
Once the end time is passed, the wallpaper from app launch is restored.  
If not set, switching continues as long as the app is running.

**Day of Week**

Sets the day of the week on which wallpaper switching runs.  
If not set, wallpapers will switch on any day while the app is running.

**Date**

Sets the day of the month on which wallpaper switching runs.  
If not set, wallpapers will switch on any date while the app is running.
